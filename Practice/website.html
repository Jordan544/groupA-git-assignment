COMPUTER SYSTEM

A Computer System refers to a system that is designed to process user data and information and provides a platform for running system software. A Computer System is divided into two parts i.e. System Software and Application Software.
System Software is software designed to co-ordinate all computer activities. system software is grouped into two i.e. Utility programms, Proggramming Languages and Operating System.
Application Software refers to programms that perform specific tasks for the user. This may include; Freeshelf software and Custom/Bespoke software. As the Evolution of computers from the Mechanical Era through the Electro-mechanical Electro-mechanical to the Electrical Era (Current) commenced. The Computer System improved heavily from Non-electrical machines which could be used to perform basic calculations i.e. Addition(+), Subtraction(-), Multiplication(*) and Division(/). to Machines that are Fast and Respond to human Languages (Artificial Intelligence) this clearly indicates that the computer has changed over time. This improvement in computer technology has come with alot advantages forexample; The work that used to take days is now done within a day and with a few people this helps the job employer to save labour money.
