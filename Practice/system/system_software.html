SYSTEM SOFTWARE

This is software that is designed to co-ordinate all activities of the computer system. System software helps the user to communicate with the computer and also helps the computer to communicate with devices (External or Internal) connected to it. External devices are connected to the computer via ports these include all peripheral devices like Keyboard, Mouse and storage devices like Hard disks, Compact disk, Memory cards etc Internal devices are devices that are connected to the computer's motherboard. these devices ars system built and the computer can not work well without them.
